# Discovery — matéria-prima para PRDs

Notas de descoberta vivem aqui: transcrições de entrevistas, notas de chamadas,
sínteses, surveys, hipóteses não confirmadas. É a *matéria-prima* — o
`discovery-researcher` destila isto em hipóteses estruturadas; o `prd-author`
usa as hipóteses, não as notas cruas, ao escrever um PRD.

## Organização sugerida

```
discovery/
  interviews/
    YYYY-MM-DD-<nome-ou-segmento>.md
  synthesis/
    YYYY-MM-DD-<tema>.md          # output do discovery-researcher
  hypotheses.md                    # backlog de hipóteses por testar
```

## Princípios

- **Cita fontes.** "User #3 disse X" não é "users querem X".
- **Distingue facto / padrão / opinião.** Marca o nível de evidência.
- **Não inventes descoberta.** Se não há dados, di-lo.
